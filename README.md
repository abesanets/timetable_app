# Расписание МГКЦТ - Android приложение (v4.1)

Android-приложение для просмотра расписания МГКЦТ: расписание групп и преподавателей, звонки, справочник сотрудников, настройки подгрупп и виджет рабочего стола с ручным и фоновым обновлением.

## Основные возможности

- **Расписание групп**: загрузка актуального недельного расписания с сайта колледжа по номеру группы.
- **Расписание преподавателей**: поиск преподавателя по фамилии с подсказками и открытие его расписания из справочника.
- **Детали занятий**: просмотр предмета, аудитории, преподавателя и данных по подгруппам в нижней панели.
- **Настройки подгруппы**: фильтрация расписания по всем подгруппам, первой или второй подгруппе.
- **Вторая подгруппа в деталях**: отдельная настройка для показа занятий другой подгруппы в карточке занятия.
- **Офлайн-доступ**: сохранение последней успешно загруженной группы и расписания через DataStore.
- **Устойчивость к ошибкам**: при проблемах с сетью или сервером приложение показывает сохраненное расписание и сообщает, что обновление не удалось.
- **Расписание звонков**: отдельный экран со звонками для учебных дней.
- **Справочник персонала**: администрация, преподаватели и сотрудники с быстрым переходом к расписанию преподавателя.
- **Виджет расписания**: показывает ближайший учебный день, учитывает выбранную подгруппу и поддерживает обновление с рабочего стола.
- **Настраиваемое обновление виджета**: ручное обновление или периодическое обновление через WorkManager от 15 минут до суток.
- **Современный интерфейс**: Jetpack Compose, Material 3, edge-to-edge, светлая и темная темы, динамические цвета Material You.

## Технологии

- **Язык**: Kotlin
- **UI**: Jetpack Compose, Material 3, Navigation Compose
- **Сеть**: OkHttp 4
- **Парсинг HTML**: Jsoup
- **Хранение данных**: DataStore Preferences + Gson
- **Фоновые задачи**: WorkManager
- **Виджеты**: Jetpack Glance + Glance Material 3
- **Архитектура**: feature-based структура модулей

## Структура проекта

```text
app/src/main/java/com/example/schedule/
├── core/        # Общие утилиты
├── data/        # Модели, сетевой слой, парсер и Preferences
├── features/    # Расписание, звонки, персонал, настройки и виджет
└── ui/          # MainActivity, навигация и тема приложения
```

## Системные требования

- **Минимальная версия Android**: API 24, Android 7.0+
- **Target SDK**: API 34
- **Compile SDK**: API 34
- **Android Studio**: Ladybug 2024.2.1 или новее
- **Интернет**: нужен для первичной загрузки расписания и дальнейших обновлений
